# Sistema de Notas Fiscais (Projeto Completo)

## 🚀 Como rodar

### Pré-requisitos
- Docker

### Rodar tudo:
```
docker-compose up --build
```

### Acessos:
- Frontend: http://localhost:4200
- Estoque API: http://localhost:5001/swagger
- Faturamento API: http://localhost:5002/swagger

## 🧪 Teste rápido
1. Cadastre produto
2. Crie nota via Swagger
3. Imprima nota

## 💡 Diferenciais
- Microsserviços
- Retry com Polly
- EF Core + SQL Server
- Dockerizado
